# Apapter Pattern

기존의 함수를 요구사항에 맞춰 수정하지 않고 
다른 인터페이스를 이용하여 재이용하는 패턴

즉, 이미 제공되어 있는 것과 필요한 것 사이의 차이를 없애주는 디자인 패턴.
